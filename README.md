# ENG-AI Refined

نسخة محسنة من موقع ENG-AI تتضمن:
- إضافة شعار وزارة التربية والتعليم الأردنية الجديد
- تحسين الأزرار والواجهة العامة
- إضافة مساعد ذكي تفاعلي
- تحسين البحث داخل بنك الأسئلة وأوراق العمل

## ملفات النشر
- index.html
- style.css
- script.js
- assets/moe-logo.jpg

## الرفع إلى GitHub Pages
1. افتح المستودع الحالي.
2. استبدل الملفات القديمة بهذه الملفات.
3. تأكد أن اسم الصفحة الرئيسية هو `index.html`.
4. ارفع مجلد `assets` كاملًا.
